# mt-website
website project based on python .expected to merge back-end tech like python, flask etc and front tech like vue...

**安装相关依赖**

`pip install -r requirement.txt`

**应用必须的环境变量**

* MAIL_USERNAME
* MAIL_PASSWORD
* SECRET_KEY（开发环境已预设）

**终端相关命令及变量**

* FLASK_APP
* FLASK_DEBUG
* `flask run`
* `flask shell`
* `flask db`
